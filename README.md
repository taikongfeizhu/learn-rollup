# Learn Rollup.js

This is an example project to accompany a tutorial on using [Rollup](http://rollupjs.org/).

## Quickstart

``` sh
# Clone the repo
git remote add origin git@github.com:taikongfeizhu/learn-rollup.git

# Move into the repo
cd learn-rollup/

# Install dependencies
npm install

# Start the watcher
npm run start

# Browser View and enjoy it
http://127.0.0.1:3001

```
